## âš™ï¸ Instalasi & Lingkungan Pengembangan

Pastikan sistem telah memenuhi prasyarat berikut:

* Node.js
* PHP
* Composer

### Clone Repositori

```bash
git clone https://github.com/blackcatin/TitikKoma.git
cd TitikKoma
```

### Instalasi Dependensi Backend

```bash
composer install
```

### Instalasi Dependensi Frontend

```bash
npm install
# atau
yarn install
```

### Konfigurasi Environment

```bash
cp .env.example .env
php artisan key:generate
```

### Menjalankan Server Pengembangan

```bash
# Laravel development server
php artisan serve

# Vite Hot Module Replacement
npm run dev
```
